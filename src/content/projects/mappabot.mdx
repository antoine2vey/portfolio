Mappabot est un bot discord permettant de créer de l'interaction entre les membres d'un serveur discord. Grâce à sa monnaie virtuelle, le bot permet de nombreuses interactions malicieuses entre les membres afin de grimper dans le classement sur une interface web, mais aussi de nombreux jeux pour parier sa monnaie customisable par serveur.
