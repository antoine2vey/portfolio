Lily est une application de soin des plantes d'intérieur, construite sous forme de monorepo TypeScript avec Bun. Le projet comprend une API backend utilisant Effect pour la programmation fonctionnelle typée, une base de données PostgreSQL avec Drizzle ORM, une application mobile React Native (Expo), un site marketing Next.js, un dashboard admin en React, ainsi qu'un serveur MCP et une base de connaissances vectorielle (pgvector) pour le RAG.

L'architecture repose entièrement sur le framework Effect, avec un système de dépendances injecté, de la gestion d'erreurs typée et du logging structuré à travers tous les packages.
