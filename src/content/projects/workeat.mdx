Workeat est un projet de licence qui développe le côté e-commerce de la livraison de nourriture pour les professionnels. Une application mobile ainsi qu'un site a été pensé pour les utilisateurs, mais aussi une application pour les livreurs pour gérer leur trajets et livraisons.
